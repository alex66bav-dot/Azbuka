# TASKS

- Буква А полностью готова.
- Для буквы А работают пять слов:
  Арбуз, Аист, Апельсин, Автобус, Акула.
- Буква Б полностью готова.
- Для буквы Б работают пять слов:
  Бабочка, Банан, Белка, Барабан, Бегемот.
- Все изображения для А и Б загружаются из соответствующих папок.
- Работает мини-игра с воздушными шариками.
- После трёх правильных попаданий игра перезапускается.
- Буква становится синей только после успешного прохождения игры.
- Прогресс сохраняется в localStorage.
- Следующая буква для наполнения: В.
- Следующий шаг: подготовить пять изображений для буквы В и проверить их в приложении.
