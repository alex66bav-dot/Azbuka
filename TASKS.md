# TASKS

- Подключены 33 буквы русского алфавита и пять слов для каждой буквы.
- В папках изображений находятся 165 иллюстраций.
- Для каждой буквы доступны пять мини-игр:
  1. Шарики.
  2. Найди два слова.
  3. Найди тень.
  4. Соедини линией.
  5. Проведи автобус.
- Для Ъ, Ь и Ы задания используют слова, содержащие нужный знак или букву.
- Каждый завершённый экран отдельно отмечается в кольце прогресса.
- Карточка буквы становится синей только после прохождения всех пяти игр.
- Общий прогресс показывается над сеткой букв и сохраняется в localStorage.
- При повторе меняются карточки, тени, изображения и маршрут автобуса.
- Похвала выбирается из пяти фраз и не повторяется два раза подряд.
- Текущий этап: пользовательское тестирование пяти игр на компьютере и телефоне.
- После тестирования: исправить только найденные неудобства, затем добавить звуки.
