# Иллюстрации букв

## Правила именования изображений

- Одна папка на букву с латинским именем: `assets/images/letters/<latin-letter>/`.
- Имя файла — название слова строчными латинскими буквами в транслитерации.
- Формат файла — `.webp`.
- Прозрачный фон в `.webp` поддерживается.

Полный шаблон пути: `assets/images/letters/<latin-letter>/<название-слова>.webp`.

## Первые иллюстрации для «А»

- `assets/images/letters/a/arbuz.webp`
- `assets/images/letters/a/aist.webp`
- `assets/images/letters/a/apelsin.webp`
- `assets/images/letters/a/avtobus.webp`
- `assets/images/letters/a/akula.webp`

После добавления файла открытая карточка автоматически проверит его и покажет иллюстрацию.
