# PROJECT

- Название проекта: Azbuka.
- Коммерческое детское приложение для изучения русского алфавита.
- Возраст: 3–6 лет.
- Платформы: браузер, Android, iOS.
- Концепция: каждая буква — отдельное маленькое приключение.
- Интерфейс должен быть максимально простым для ребёнка.
- Один экран — одна задача.
- Большие кнопки, минимум текста, плавные анимации.
- Первая версия содержит русский алфавит.
- В будущем: цифры, цвета, животные, формы и другие разделы.
- Приложение связано с детским YouTube-каналом и должно использовать единый стиль персонажей.
